Prework Study Guide

HTML

  • Use 'section' to define different sections
  • 'head' contains 'meta' and 'title'
  • 'body' holds the content of page aside from metadata
  • 'main' wraps the main content of a page
  • 'header' and 'footer' provides the top and bottom of the page
  • 'p' indicates where a paragraph is
  • 'ol' gives ordered lists; 'ul' gives unordered
  • Populate lists with 'li' elements

Git

  • All git commands start with $ git
  • 'git checkout' moves branch. New branch with '-b *feature/title*'
  • 'git status' shows current status
  • 'git add -A' primes for adding content
  • 'git commit' commits content. 'm "message here"' to add title to commit
  • 'git pull origin main' ensures local branch is synced with repo
  • 'git push origin *feature/title*' to push to repo

GitBash

  • 'cd' changes directory
  • 'cd ..' goes backwards
  • 'ls' lists current directory
  • 'mkdir' makes directory
  • 'code .' opens VS Code in current location
  • 'pwd' prints current directory
  • 'touch' - makes a new file
  • 'cp' copies a file
  • 'mv' moves a file or directory
